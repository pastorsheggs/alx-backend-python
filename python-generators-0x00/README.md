# 📦 Python Generators – Getting Started

This project introduces the use of **Python generators** to handle data efficiently — especially with large datasets like database rows. The script also demonstrates how to interact with a **MySQL database**, create a schema, and populate it from a CSV file.

---

## 📌 Objective

- Set up a MySQL database called `ALX_prodev`
- Create a table called `user_data` with the following fields:
  - `user_id` (UUID, Primary Key, Indexed)
  - `name` (VARCHAR, NOT NULL)
  - `email` (VARCHAR, NOT NULL)
  - `age` (DECIMAL, NOT NULL)
- Insert sample data from `user_data.csv`
